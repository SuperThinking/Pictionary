# Pictionary
Pictionary Game - Using React Native/NodeJS/Socket.io

How to run:
1) Clone this repo

2) Run Backend:<br/>
```
  cd pictionary-server
  npm install
  npm start
```
3) In a different terminal,
```
  cd PictionaryClient/
  npm start
  npm run android
```
##### Currently no DB is being used and everything is being done in memory (Reload the backend and whooosh)
